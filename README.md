OSS Academy Pro — Guia Interativo e Diário de Jiu-Jitsu

O OSS Academy Pro é uma aplicação web interativa e educativa desenvolvida para apoiar e acompanhar praticantes de Jiu-Jitsu (a "Arte Suave") na sua evolução desportiva. Com foco na disciplina, respeito mútuo e desenvolvimento pessoal, a aplicação serve como um companheiro de treino digital para atletas de todas as idades.

Este projeto combina teoria desportiva, estudo de regras de arbitragem e uma ferramenta prática de monitorização de treinos diários.

🚀 Funcionalidades Principais

1. 🥋 Jornada de Graduação (Faixas)

Um mapa interativo que guia o utilizador através das diferentes etapas do Jiu-Jitsu (Faixa Branca à Faixa Preta).

Foco Técnico: Explicação do objetivo de cada faixa (sobrevivência, defesa, transições, fluidez e maestria).

Dicas Práticas: Conselhos construtivos para ajudar a focar nos aspetos corretos em cada fase do treino.

2. 📖 Dicionário de Golpes (Enciclopédia)

Uma biblioteca digital segura para consulta de posições e finalizações clássicas.

Filtros por Posição: Encontra rapidamente técnicas a partir da Guarda Fechada, Montada ou Pegada de Costas.

Fichas Técnicas: Inclui o nome em Português e Inglês, a mecânica detalhada do movimento focado na execução segura e o nível de dificuldade estimado.

3. 🧮 Simulador de Combate & Tabela de Pontos

Uma ferramenta de aprendizagem interativa sobre as regras oficiais de pontuação.

Painel Dinâmico: Simulação de pontos em tempo real para dois atletas, com registo automático de histórico de ações.

Regras Didáticas: Destaque para a regra essencial dos 3 segundos de estabilização para validação dos pontos.

4. 📝 Diário de Treinos Pessoal

Uma secção prática para registo e reflexão sobre a consistência dos treinos.

Histórico de Aulas: Permite registar a data, o tipo de aula (técnico, sparring, sem kimono), duração e energia/humor do dia.

Foco do Dia: Campo para anotações pessoais sobre o que correu bem e o que pode ser melhorado na próxima sessão.

🛠️ Tecnologias Utilizadas

A aplicação foi desenvolvida utilizando tecnologias modernas de desenvolvimento web:

React (Biblioteca JavaScript para criação de interfaces dinâmicas)

Tailwind CSS (Estilização rápida, moderna e responsiva)

Lucide Icons (Pacote de ícones minimalistas e intuitivos)

📦 Como Executar o Projeto Localmente

Para correres este projeto no teu computador, segue os passos abaixo:

Clona o repositório:

git clone [https://github.com/teu-utilizador/oss-academy-pro.git](https://github.com/teu-utilizador/oss-academy-pro.git)


Navega até à pasta do projeto:

cd oss-academy-pro


Instala as dependências:

npm install


Inicia o servidor de desenvolvimento:

npm start


Abre http://localhost:3000 no teu navegador para veres a aplicação a funcionar!

O Jiu-Jitsu é muito mais do que técnicas de defesa pessoal; é uma filosofia de vida que constrói o caráter através da persistência. O OSS Academy Pro foi criado para incentivar:

Respeito: Valorizar os colegas de treino e professores.

Saúde e Bem-estar: Promover um estilo de vida ativo, equilibrado e saudável.

Superação: Aprender com os erros e celebrar as pequenas vitórias de cada treino.
